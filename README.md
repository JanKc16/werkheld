# 🔧 werkheld – Dein Handwerker in Minuten

MVP-Plattform für Handwerkervermittlung. Landing Page + vollständige App mit Registrierung, Dashboards und Auftragsverwaltung.

## Schnellstart

### 1. Node.js installieren (falls nicht vorhanden)
→ https://nodejs.org (LTS-Version herunterladen und installieren)

### 2. Terminal in VS Code öffnen
- VS Code öffnen
- Diesen Ordner öffnen: `Datei → Ordner öffnen → werkheld-project`
- Terminal öffnen: `Terminal → Neues Terminal` (oder `Strg+Ö`)

### 3. Abhängigkeiten installieren
```bash
npm install
```

### 4. Starten
```bash
npm run dev
```

### 5. Im Browser öffnen
→ http://localhost:5173

Fertig! 🎉

## Auf GitHub pushen

```bash
git init
git add .
git commit -m "werkheld MVP"
```

Dann auf https://github.com/new ein Repository erstellen und:

```bash
git remote add origin https://github.com/DEIN-USERNAME/werkheld.git
git branch -M main
git push -u origin main
```

## Live deployen (kostenlos)

1. https://vercel.com → Mit GitHub anmelden
2. "Add New Project" → Repository auswählen
3. "Deploy" klicken
4. Fertig → Du bekommst eine URL wie `werkheld.vercel.app`
